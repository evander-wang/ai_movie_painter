# Domain Glossary

## Workflow

An AI video production process made of connected nodes. A workflow describes how prompts, references, generation steps, processing tools, and output steps relate to each other.

## Canvas

The spatial workspace where a workflow is arranged, inspected, imported, exported, and edited.

## Node

A workflow item placed on the canvas. A node can represent text, image reference, video generation, audio processing, or a tool step.

## Edge

A directed connection between two nodes. An edge means the source node contributes input, reference, constraints, or output to the target node.

## Handle

The connection point on a node used to create or receive an edge.

## Viewport

The current canvas camera state: horizontal offset, vertical offset, and zoom.

## Attribute Panel

The contextual panel shown for a selected node. It exposes the node's editable or inspectable properties.

## Node Catalog

The reviewed list of node types that can be added to a workflow, including their labels, categories, kinds, and default attribute metadata.

## Default Canvas

The initial workflow shown to users when the editor opens without imported canvas data.

## Import/Export Schema

The versioned JSON shape used to persist or exchange a canvas.

## Active Path

The set of edges connected to the selected node. Active path edges may receive stronger visual emphasis than inactive edges.
