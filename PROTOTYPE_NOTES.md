# Canvas UI Prototype Notes

Question: how close can we recreate the LibTV-style infinite canvas UI, color system, node cards, bottom toolbars, and expandable panels?

Answer so far: the prototype uses `@xyflow/react`, matching the inspected product's React Flow / XYFlow structure. It focuses on visual and interaction fidelity: dark dotted canvas, DOM-rendered media/text/group nodes, animated edges, floating icon toolbars, a right drawer, node-add popover, zoom menu, and keyboard shortcuts modal.

This is throwaway prototype code. Keep or delete it after deciding which parts should move into production.
